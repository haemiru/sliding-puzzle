# 부록 B. 향기 선택 가이드

수면에 도움이 되는 에센셜 오일 10가지의 특성, 효과, 주의사항을 정리했습니다.

---

## 추천 순위별 에센셜 오일

### 1순위 — 처음 시작할 때 추천

#### 1. 라벤더 (Lavender)
| 항목 | 내용 |
|------|------|
| **학명** | Lavandula angustifolia |
| **향 특성** | 부드럽고 깨끗한 꽃 향 |
| **주요 효과** | 진정, 이완, 수면 유도, 불안 감소 |
| **추천 대상** | 모든 아이 (가장 범용적) |
| **디퓨저 사용량** | 2~3방울 |
| **주의사항** | 과량 사용 시 두통 가능 |

#### 2. 로만 캐모마일 (Roman Chamomile)
| 항목 | 내용 |
|------|------|
| **학명** | Chamaemelum nobile |
| **향 특성** | 달콤하고 따뜻한 사과 같은 향 |
| **주요 효과** | 깊은 진정, 불안 감소, 감정 안정 |
| **추천 대상** | 불안감이 높은 아이 |
| **디퓨저 사용량** | 2~3방울 |
| **주의사항** | 국화과 알레르기 있는 경우 주의 |

#### 3. 만다린 (Mandarin)
| 항목 | 내용 |
|------|------|
| **학명** | Citrus reticulata |
| **향 특성** | 상큼하고 달콤한 감귤 향 |
| **주요 효과** | 기분 전환, 가벼운 진정, 스트레스 감소 |
| **추천 대상** | 향기에 거부감이 있는 아이 (가장 잘 받아들이는 향) |
| **디퓨저 사용량** | 2~3방울 |
| **주의사항** | 광독성 있음 — 피부 사용 후 자외선 노출 금지 (디퓨저 사용은 무관) |

---

### 2순위 — 기본 향에 익숙해진 후

#### 4. 시더우드 (Cedarwood)
| 항목 | 내용 |
|------|------|
| **학명** | Cedrus atlantica |
| **향 특성** | 따뜻하고 나무 향 |
| **주요 효과** | 안정감, 수면 유도, 그라운딩 |
| **추천 대상** | 나무 향을 좋아하는 아이, 감각 둔감형 |
| **디퓨저 사용량** | 1~2방울 |
| **주의사항** | 임산부 사용 금지 |

#### 5. 베르가못 (Bergamot)
| 항목 | 내용 |
|------|------|
| **학명** | Citrus bergamia |
| **향 특성** | 상큼하면서도 달콤한 시트러스 향 |
| **주요 효과** | 스트레스 감소, 기분 개선, 가벼운 진정 |
| **추천 대상** | 감정 기복이 큰 아이 |
| **디퓨저 사용량** | 2~3방울 |
| **주의사항** | 광독성 있음 (디퓨저 사용은 안전) |

#### 6. 일랑일랑 (Ylang Ylang)
| 항목 | 내용 |
|------|------|
| **학명** | Cananga odorata |
| **향 특성** | 달콤하고 꽃향이 강한 향 |
| **주요 효과** | 심박수 감소, 깊은 이완, 혈압 안정 |
| **추천 대상** | 과각성 상태인 아이 |
| **디퓨저 사용량** | 1방울 (향이 강함) |
| **주의사항** | 과량 시 두통·구역질 가능, 소량 사용 |

#### 7. 스위트 오렌지 (Sweet Orange)
| 항목 | 내용 |
|------|------|
| **학명** | Citrus sinensis |
| **향 특성** | 밝고 달콤한 오렌지 향 |
| **주요 효과** | 기분 개선, 가벼운 진정, 식욕 조절 |
| **추천 대상** | 감귤 향을 좋아하는 아이 |
| **디퓨저 사용량** | 2~3방울 |
| **주의사항** | 가장 순한 오일 중 하나 |

---

### 3순위 — 블렌딩용

#### 8. 프랑킨센스 (Frankincense)
| 항목 | 내용 |
|------|------|
| **향 특성** | 따뜻하고 수지 같은 깊은 향 |
| **주요 효과** | 깊은 이완, 명상적 분위기, 호흡 안정 |
| **디퓨저 사용량** | 1~2방울 |

#### 9. 마조람 (Marjoram)
| 항목 | 내용 |
|------|------|
| **향 특성** | 따뜻하고 허브 향 |
| **주요 효과** | 근육 이완, 수면 유도 |
| **디퓨저 사용량** | 1~2방울 |

#### 10. 클라리세이지 (Clary Sage)
| 항목 | 내용 |
|------|------|
| **향 특성** | 허브와 약간 달콤한 향 |
| **주요 효과** | 깊은 이완, 스트레스 해소 |
| **디퓨저 사용량** | 1~2방울 |
| **주의사항** | 간질 병력 있는 경우 사용 금지 |

## 추천 블렌딩 레시피

| 이름 | 구성 | 효과 | 추천 상황 |
|------|------|------|-----------|
| 달콤한 꿈 | 라벤더 2 + 만다린 1 | 편안한 입면 | 처음 시작할 때 |
| 깊은 숲 | 시더우드 1 + 라벤더 1 | 깊은 이완 | 감각 둔감형 아이 |
| 평온한 밤 | 캐모마일 2 + 라벤더 1 | 불안 감소 | 불안감 높은 아이 |
| 과일 잠 | 만다린 2 + 오렌지 1 | 기분 개선 + 진정 | 향기 거부가 있는 아이 |
| 따뜻한 안식 | 라벤더 1 + 프랑킨센스 1 | 명상적 이완 | 루틴이 안정된 후 |

> - 디퓨저 사용 시 하루 30분 이내, 반드시 환기와 함께
> - 피부에 원액을 직접 바르지 마세요
> - 아이 손이 닿지 않는 곳에 보관하세요
> - 새 오일은 반드시 패치 테스트 후 사용하세요
> - 3세 미만 영유아에게는 사용하지 마세요
