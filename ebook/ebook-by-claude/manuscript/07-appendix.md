# 부록: 템플릿 & 체크리스트

<div class="chapter-desc">전자책 제작에 바로 활용할 수 있는 프롬프트 템플릿과 체크리스트 모음.</div>
</div>

## A. 프롬프트 템플릿 모음

### A-1. 주제 발굴 프롬프트

<div class="box-prompt">

<pre>
[분야]에서 전자책으로 만들 수 있는 수익성 높은 주제 10개를 추천해줘.

각 주제별로:
- 타겟 독자
- 예상 분량 (페이지 수)
- 경쟁 강도 (높음/중간/낮음)
- 수익 잠재력 (높음/중간/낮음)
- 차별화 아이디어 1개

표 형식으로 정리해줘.
</pre>

</div>

### A-2. 목차 생성 프롬프트

<div class="box-prompt">

<pre>
다음 전자책의 상세 목차를 만들어줘.

제목: [제목]
타겟: [대상 독자]
분량: 약 [N]페이지
유형: [가이드/워크북/사례집]

요구사항:
- 6~8개 챕터
- 각 챕터에 3~4개 섹션
- 각 챕터 끝에 워크시트 포함
- 챕터별 예상 페이지 수
- 핵심 메시지 한 줄
</pre>

</div>

### A-3. 본문 작성 프롬프트

<div class="box-prompt">

<pre>
[마스터 프롬프트]를 적용한 상태에서,
Chapter [N]의 "[섹션 제목]" 섹션을 작성해줘.

섹션 목표: [독자가 이 섹션에서 배울 것]

포함 요소:
- 도입 (왜 중요한지)
- 핵심 내용 (단계별 설명)
- 구체적 예시 1~2개
- 실행 팁

분량: 약 [N]자
</pre>

</div>

### A-4. 교정 프롬프트

<div class="box-prompt">

<pre>
다음 글을 전자책 수준으로 교정해줘.

[원고 텍스트]

교정 기준:
1. 맞춤법/문법 오류 수정
2. 문장 길이 최적화 (30자 이내)
3. 존댓말 통일 (합니다체)
4. 수동태 → 능동태
5. 불필요한 수식어 제거
6. 문단 길이 최적화 (3~5문장)

변경 사항을 목록으로 알려줘.
</pre>

</div>

### A-5. 마케팅 콘텐츠 프롬프트

<div class="box-prompt">

<pre>
전자책 "[제목]"의 마케팅 콘텐츠를 만들어줘.

타겟: [대상 독자]
핵심 가치: [독자가 얻는 것]
판매 가격: [가격]원

만들어줄 것:
1. 크몽 상품 설명 (상세, 1500자)
2. SNS 홍보 문구 3개 (200자 이내)
3. 블로그 리뷰 요청 메시지 (300자)
4. 런칭 할인 이벤트 공지 (300자)
</pre>

</div>

## B. 전체 프로세스 체크리스트

### B-1. 기획 단계

- [ ] 주제 3개 이상 후보 선정
- [ ] 경쟁 전자책 5권 이상 분석
- [ ] 최종 주제 확정 및 차별화 포인트 정리
- [ ] 타겟 독자 페르소나 작성
- [ ] 목차 초안 작성
- [ ] 분량 및 가격 결정
- [ ] 일정 계획 수립

### B-2. 작성 단계

- [ ] 마스터 프롬프트 설정
- [ ] 챕터별 초안 작성 완료
- [ ] 개인 경험/사례 추가
- [ ] 워크시트/실습 자료 작성
- [ ] 이미지/표/도표 제작

### B-3. 편집 단계

- [ ] 1차 편집 (구조) 완료
- [ ] 2차 편집 (내용) 완료
- [ ] 3차 교정 (문장) 완료
- [ ] 맞춤법 검사 도구 활용
- [ ] 제3자 피드백 반영

### B-4. 디자인 단계

- [ ] CSS 스타일시트 설정
- [ ] 표지 디자인 완료
- [ ] 본문 레이아웃 확인
- [ ] 이미지 해상도 확인 (150 DPI 이상)
- [ ] PDF 빌드 및 점검

### B-5. 출간 단계

- [ ] 판매 플랫폼 계정 생성
- [ ] 상품 등록 (제목, 설명, 썸네일)
- [ ] 가격 설정 및 런칭 할인 적용
- [ ] 베타 리더 리뷰 등록
- [ ] 마케팅 콘텐츠 배포
- [ ] 출간 후 1주일 판매 데이터 분석
