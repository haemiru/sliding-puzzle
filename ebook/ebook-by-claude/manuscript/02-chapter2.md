# 수익이 되는 주제 찾기

<div class="chapter-desc">시장 조사 방법과 수익성 높은 니치를 선정하는 전략을 배웁니다.</div>
</div>

## 좋은 주제의 3가지 조건

전자책의 성패는 주제 선정에서 80%가 결정됩니다. 좋은 주제는 다음 세 가지 조건을 모두 만족해야 합니다:

1. **수요가 있는가?** — 사람들이 실제로 검색하고 돈을 쓰는 주제
2. **당신이 쓸 수 있는가?** — 경험, 지식, 또는 충분한 리서치가 가능한 주제
3. **경쟁에서 이길 수 있는가?** — 차별화 포인트가 있는 주제

<div class="box-key">

**수익 공식:** 수요(검색량) × 전환율(구매 의향) × 차별화(경쟁 우위) = 수익 잠재력

</div>

## 수요 조사 방법

### 1단계: 키워드 리서치

검색 엔진에서 사람들이 어떤 정보를 찾고 있는지 확인하세요.

<div class="box-prompt">

**Claude 프롬프트:**

<pre>
다음 분야에서 전자책으로 만들기 좋은 주제 20개를 추천해줘.

분야: [자기계발 / 재테크 / 건강 / 기술 / 취미]

각 주제별로 다음을 분석해줘:
1. 예상 타겟 독자
2. 검색 수요 (높음/중간/낮음)
3. 경쟁 강도 (높음/중간/낮음)
4. 수익화 가능성 (높음/중간/낮음)

표 형식으로 정리해줘.
</pre>

</div>

### 2단계: 경쟁 분석

크몽, 탈잉, 클래스101, 아마존 킨들에서 비슷한 주제의 전자책을 조사합니다.

| 분석 항목 | 확인 내용 |
|-----------|----------|
| 가격대 | 경쟁 전자책의 평균 가격 |
| 리뷰 수 | 판매량의 간접 지표 |
| 리뷰 내용 | 독자가 좋아하는 점 / 아쉬운 점 |
| 목차 구성 | 어떤 내용을 다루는지 |
| 페이지 수 | 분량 기준 |

### 3단계: 수익성 높은 니치 카테고리

<div class="box-tip">

**검증된 수익 니치 TOP 5:**

1. **실용 기술 가이드** — "노코드로 앱 만들기", "엑셀 자동화" 등
2. **자기계발 + 실행법** — "아침 루틴 설계", "습관 형성 30일" 등
3. **부업/재테크** — "월 100만원 부수입", "투자 입문서" 등
4. **육아/교육** — "영어 독서 습관", "초등 수학 학습법" 등
5. **건강/웰빙** — "수면 개선", "홈트레이닝 가이드" 등

</div>

## 주제 검증 체크리스트

아래 항목 중 **5개 이상**에 해당하면 좋은 주제입니다:

- [ ] 해당 주제로 월 1,000건 이상 검색이 발생한다
- [ ] 크몽/클래스101에 유사 상품이 3개 이상 있다 (수요 증명)
- [ ] 하지만 10개 미만이다 (과당경쟁 아님)
- [ ] 기존 경쟁 전자책의 리뷰에 "부족한 점"이 보인다
- [ ] 내가 차별화할 수 있는 포인트가 2개 이상 있다
- [ ] 해당 분야에 6개월 이상 경험이 있다
- [ ] 전자책 가격을 15,000원 이상 받을 수 있다
- [ ] 후속 상품(강의, 컨설팅) 연계가 가능하다

<div class="box-warning">

**피해야 할 주제:**
- 너무 광범위한 주제 (예: "마케팅의 모든 것")
- 무료 정보로 충분한 주제 (예: "구글 검색 방법")
- 전문 자격이 필요한 주제 (예: "질병 치료법")
- 시의성이 지나간 주제 (예: "2024년 트렌드")

</div>
