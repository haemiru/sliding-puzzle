# 시작하기 전에

## 왜 지금 전자책인가?

전자책 시장은 매년 성장하고 있습니다. 특히 AI 도구의 등장으로 전자책 제작의 진입 장벽이 크게 낮아졌습니다. 더 이상 전문 작가가 아니어도, 디자이너가 아니어도 양질의 전자책을 만들 수 있는 시대가 되었습니다.

### 전자책 시장의 현재

| 구분 | 수치 | 비고 |
|------|------|------|
| 글로벌 전자책 시장 규모 | 약 200억 달러 | 2025년 기준 |
| 연평균 성장률 | 4.9% | 2025~2030 전망 |
| 한국 전자책 이용률 | 23.4% | 성인 기준 |
| 셀프 퍼블리싱 비중 | 30%+ | 아마존 킨들 기준 |

### AI가 바꾸는 전자책 제작

<div class="box-key">

**핵심 변화 3가지:**

1. **제작 시간 단축**: 기존 3~6개월 → AI 활용 시 2~4주
2. **전문성 장벽 해소**: 글쓰기 경험이 없어도 구조화된 콘텐츠 작성 가능
3. **비용 절감**: 편집, 교정, 디자인 비용 대폭 절감

</div>

# 수익이 되는 주제 찾기

## 좋은 주제의 3가지 조건

전자책의 성패는 주제 선정에서 80%가 결정됩니다. 좋은 주제는 다음 세 가지 조건을 모두 만족해야 합니다:

1. **수요가 있는가?** — 사람들이 실제로 검색하고 돈을 쓰는 주제
2. **당신이 쓸 수 있는가?** — 경험, 지식, 또는 충분한 리서치가 가능한 주제
3. **경쟁에서 이길 수 있는가?** — 차별화 포인트가 있는 주제

<div class="box-key">

**수익 공식:** 수요(검색량) × 전환율(구매 의향) × 차별화(경쟁 우위) = 수익 잠재력

</div>

## 수요 조사 방법

### 1단계: 키워드 리서치

검색 엔진에서 사람들이 어떤 정보를 찾고 있는지 확인하세요.

<div class="box-prompt">

**Claude 프롬프트:**

<pre>
다음 분야에서 전자책으로 만들기 좋은 주제 20개를 추천해줘.

분야: [자기계발 / 재테크 / 건강 / 기술 / 취미]

각 주제별로 다음을 분석해줘:
1. 예상 타겟 독자
2. 검색 수요 (높음/중간/낮음)
3. 경쟁 강도 (높음/중간/낮음)
4. 수익화 가능성 (높음/중간/낮음)

표 형식으로 정리해줘.
</pre>

</div>

### 2단계: 경쟁 분석

크몽, 탈잉, 클래스101, 아마존 킨들에서 비슷한 주제의 전자책을 조사합니다.

| 분석 항목 | 확인 내용 |
|-----------|----------|
| 가격대 | 경쟁 전자책의 평균 가격 |
| 리뷰 수 | 판매량의 간접 지표 |
| 리뷰 내용 | 독자가 좋아하는 점 / 아쉬운 점 |
| 목차 구성 | 어떤 내용을 다루는지 |
| 페이지 수 | 분량 기준 |

### 3단계: 수익성 높은 니치 카테고리

<div class="box-tip">

**검증된 수익 니치 TOP 5:**

1. **실용 기술 가이드** — "노코드로 앱 만들기", "엑셀 자동화" 등
2. **자기계발 + 실행법** — "아침 루틴 설계", "습관 형성 30일" 등
3. **부업/재테크** — "월 100만원 부수입", "투자 입문서" 등
4. **육아/교육** — "영어 독서 습관", "초등 수학 학습법" 등
5. **건강/웰빙** — "수면 개선", "홈트레이닝 가이드" 등

</div>

## 전자책의 기본 구조

잘 팔리는 전자책에는 공통된 구조가 있습니다. 독자가 처음부터 끝까지 자연스럽게 읽을 수 있는 흐름을 만들어야 합니다.

### 표준 전자책 구조

```
1. 표지
2. 저작권 / 면책 조항
3. 목차
4. 프롤로그 (왜 이 책을 썼는가)
5. 본문 챕터 (5~10개)
   - 도입 → 이론 → 실습 → 정리
6. 에필로그 (다음 단계 안내)
7. 부록 (템플릿, 체크리스트)
8. 저자 소개
```

### 적정 분량 가이드

| 전자책 유형 | 페이지 수 | 단어 수 (한글) | 가격대 |
|-------------|-----------|---------------|--------|
| 미니 가이드 | 20~40p | 8,000~15,000자 | 5,000~9,900원 |
| 표준 전자책 | 50~100p | 20,000~40,000자 | 9,900~19,900원 |
| 프리미엄 가이드 | 100~200p | 40,000~80,000자 | 19,900~39,900원 |
| 종합 핸드북 | 200p+ | 80,000자+ | 29,900원+ |

<div class="box-tip">

처음이라면 **50~80페이지 분량**을 목표로 하세요. 너무 짧으면 가치를 느끼기 어렵고, 너무 길면 완성하기 어렵습니다.

</div>

### 목차 검증 포인트

목차를 만들었다면 다음을 확인하세요:

1. **독자 여정이 자연스러운가?** — 모르는 상태 → 아는 상태 → 할 수 있는 상태
2. **각 챕터가 독립적으로도 가치 있는가?** — 하나만 읽어도 배움이 있어야 함
3. **실습이 포함되어 있는가?** — 읽기만 하는 책은 가치가 떨어짐
4. **분량이 균형 잡혀 있는가?** — 한 챕터가 너무 길거나 짧지 않게

Claude와 효과적으로 작업하려면 **좋은 지시(프롬프트)**를 줄 수 있어야 합니다. 프롬프트의 품질이 곧 결과물의 품질입니다.

### 좋은 프롬프트의 5요소

| 요소 | 설명 | 예시 |
|------|------|------|
| **역할(Role)** | Claude에게 부여할 전문가 역할 | "너는 10년 경력의 재테크 전문가야" |
| **맥락(Context)** | 배경 정보와 상황 | "초보 투자자를 위한 전자책을 쓰고 있어" |
| **지시(Task)** | 구체적인 작업 내용 | "Chapter 3의 '분산 투자' 섹션을 써줘" |
| **형식(Format)** | 출력 형태 | "마크다운 형식, 2000자 내외, 예시 포함" |
| **제약(Constraint)** | 주의 사항 | "전문 용어는 쉽게 풀어 설명해줘" |

## A. 프롬프트 템플릿 모음

### A-1. 주제 발굴 프롬프트

<div class="box-prompt">

<pre>
[분야]에서 전자책으로 만들 수 있는 수익성 높은 주제 10개를 추천해줘.

각 주제별로:
- 타겟 독자
- 예상 분량 (페이지 수)
- 경쟁 강도 (높음/중간/낮음)
- 수익 잠재력 (높음/중간/낮음)
- 차별화 아이디어 1개

표 형식으로 정리해줘.
</pre>

</div>

### A-2. 목차 생성 프롬프트

<div class="box-prompt">

<pre>
다음 전자책의 상세 목차를 만들어줘.

제목: [제목]
타겟: [대상 독자]
분량: 약 [N]페이지
유형: [가이드/워크북/사례집]

요구사항:
- 6~8개 챕터
- 각 챕터에 3~4개 섹션
- 각 챕터 끝에 워크시트 포함
- 챕터별 예상 페이지 수
- 핵심 메시지 한 줄
</pre>

</div>

### A-3. 본문 작성 프롬프트

<div class="box-prompt">

<pre>
[마스터 프롬프트]를 적용한 상태에서,
Chapter [N]의 "[섹션 제목]" 섹션을 작성해줘.

섹션 목표: [독자가 이 섹션에서 배울 것]

포함 요소:
- 도입 (왜 중요한지)
- 핵심 내용 (단계별 설명)
- 구체적 예시 1~2개
- 실행 팁

분량: 약 [N]자
</pre>

</div>

### A-4. 교정 프롬프트

<div class="box-prompt">

<pre>

[원고 텍스트]

교정 기준:
1. 맞춤법/문법 오류 수정

3. 존댓말 통일 (합니다체)
4. 수동태 → 능동태
5. 불필요한 수식어 제거

변경 사항을 목록으로 알려줘.
</pre>
