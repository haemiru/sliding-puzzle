# 바이브 코딩 실습 체크리스트

**"바이브 코딩으로 1인 사업가 되기" 부록 자료**

> 이 체크리스트는 전자책의 내용을 실제로 실행에 옮기기 위한 단계별 가이드입니다.
> 위에서부터 순서대로 하나씩 체크해 나가세요.

---

## Phase 1. 마인드셋 준비 (Part 1 — 1~2일)

### Step 1. 바이브 코딩 이해하기
- [ ] 바이브 코딩이 뭔지 한 문장으로 설명할 수 있다
- [ ] 전통적 코딩과 바이브 코딩의 차이를 이해했다
- [ ] "코딩을 몰라도 된다"는 사실을 받아들였다

### Step 2. 내 강점 파악하기
- [ ] 나의 경력/경험에서 강점 3가지를 적었다
- [ ] 내가 잘 아는 분야(전문 지식)를 1개 이상 정했다
- [ ] "나이와 경험이 오히려 장점"이라는 것을 이해했다

### Step 3. 아이디어 발견하기
- [ ] 매일 겪는 불편함 5가지를 적었다
- [ ] 그중 서비스로 만들 수 있는 아이디어를 3개 골랐다
- [ ] 가장 만들고 싶은 아이디어 1개를 선택했다

**Phase 1 완료 기준:** 만들고 싶은 아이디어가 1개 이상 있다

---

## Phase 2. 도구 익히기 (Part 2 — 3~5일)

### Step 4. AI 도구 설치하기
- [ ] Claude (claude.ai) 가입 완료
- [ ] Cursor (cursor.com) 설치 완료
- [ ] AI에게 첫 인사를 건네봤다 ("안녕, 나는 코딩 초보야")

### Step 5. AI와 대화 연습하기
- [ ] 일상적인 질문을 3개 이상 해봤다
- [ ] "더 쉽게 설명해줘"라고 다시 요청해봤다
- [ ] "예를 들어 설명해줘"라고 요청해봤다
- [ ] AI가 친구처럼 편하게 느껴지기 시작했다

### Step 6. 첫 웹페이지 만들기
- [ ] AI에게 자기소개 웹페이지를 요청했다
- [ ] 결과물을 확인했다 (브라우저에서 열어봤다)
- [ ] 수정 사항을 2가지 이상 요청해봤다
- [ ] 최종 결과물이 마음에 든다

### Step 7. 인터넷에 배포하기
- [ ] Netlify 가입 완료
- [ ] 내 웹페이지를 Netlify에 업로드했다
- [ ] 나만의 URL이 생겼다 (https://______.netlify.app)
- [ ] 가족이나 친구 1명에게 링크를 공유했다
- [ ] 상대방의 반응을 확인했다

**Phase 2 완료 기준:** 내 웹페이지가 인터넷에 공개되어 있다

---

## Phase 3. 실전 프로젝트 (Part 3 — 2~4주)

### Step 8. 프로젝트 기획하기
- [ ] 서비스를 한 줄로 설명할 수 있다 ("이것은 ___를 위한 ___이다")
- [ ] 핵심 기능 3가지를 정했다 (MVP)
- [ ] 첫 번째 사용자가 누구인지 정했다
- [ ] 1페이지 기획서를 작성했다

### Step 9. 기능 만들기
- [ ] 첫 번째 핵심 기능을 AI에게 요청했다
- [ ] 첫 번째 기능이 정상 동작한다
- [ ] 두 번째 핵심 기능을 추가했다
- [ ] 세 번째 핵심 기능을 추가했다
- [ ] 에러가 발생했을 때 AI에게 해결을 요청해봤다

### Step 10. 디자인 다듬기
- [ ] 메인 색상을 정했다 (최대 3가지)
- [ ] 글꼴을 읽기 쉬운 것으로 설정했다
- [ ] 요소 사이 여백을 충분히 넣었다
- [ ] 모바일에서 테스트했다 (반응형 확인)
- [ ] 전체적으로 깔끔한 느낌이 든다

### Step 11. 피드백 받기
- [ ] 주변 사람 5명에게 보여줬다
- [ ] 3가지 질문을 했다:
  - "뭔지 바로 알겠어?"
  - "불편한 점 있어?"
  - "다시 쓰고 싶어?"
- [ ] 피드백을 "바로 수정 / 나중에 / 무시"로 분류했다
- [ ] "바로 수정" 항목을 반영했다
- [ ] 수정 후 다시 배포했다

**Phase 3 완료 기준:** MVP가 배포되어 있고, 5명의 피드백을 반영했다

---

## Phase 4. 수익화 (Part 4 — 2~4주)

### Step 12. 수익 모델 정하기
- [ ] 4가지 수익 모델을 이해했다 (광고/구독/일회성/프리미엄)
- [ ] 내 서비스에 맞는 수익 모델 1개를 선택했다
- [ ] 비슷한 서비스의 가격을 3개 이상 조사했다
- [ ] 내 서비스의 가격을 정했다

### Step 13. 결제 연동하기
- [ ] 결제 수단을 선택했다 (Stripe / 토스페이먼츠)
- [ ] AI에게 결제 기능 추가를 요청했다
- [ ] 테스트 결제가 정상 동작한다
- [ ] 결제 완료 후 흐름이 자연스럽다 (감사 페이지 등)

### Step 14. 첫 고객 확보하기
- [ ] 첫 고객 후보 3명을 정했다
- [ ] 무료 체험을 제안했다
- [ ] 첫 번째 유료 고객이 생겼다!
- [ ] 고객에게 "왜 구매했는지" 물어봤다

### Step 15. 마케팅 시작하기
- [ ] 주력 SNS 채널 1개를 정했다 (인스타/블로그/유튜브/트위터)
- [ ] 계정을 만들고 프로필을 완성했다
- [ ] 첫 번째 게시물을 올렸다
- [ ] 관련 커뮤니티 2개에 가입했다
- [ ] 커뮤니티에서 도움이 되는 댓글을 3개 이상 남겼다
- [ ] 주 2~3회 게시물 올리기를 시작했다

**Phase 4 완료 기준:** 첫 유료 고객이 있고, 마케팅 채널이 운영 중이다

---

## Phase 5. 지속 성장 (Part 5 — 계속)

### Step 16. 자동화하기
- [ ] 반복하는 작업 목록을 적었다
- [ ] 환영 이메일 자동 발송을 설정했다
- [ ] 자주 묻는 질문 5개에 대한 답변을 정리했다
- [ ] 고객 응대 챗봇을 추가했다 (또는 FAQ 페이지)
- [ ] 서비스 모니터링/알림을 설정했다
- [ ] 자동 백업을 설정했다

### Step 17. 커뮤니티 만들기
- [ ] 커뮤니티 플랫폼을 정했다 (카카오톡 오픈채팅 / 디스코드)
- [ ] 커뮤니티를 개설했다
- [ ] 운영 규칙 3가지를 정했다
- [ ] 초기 멤버 5명을 모았다
- [ ] 주간 정기 이벤트를 시작했다 (예: 월요일 목표 공유)

### Step 18. 장기 비전 세우기
- [ ] 12개월 로드맵을 작성했다
- [ ] 하루 작업 시간을 정했다 (2~3시간 권장)
- [ ] 주 1회 완전한 휴식일을 정했다
- [ ] 건강 루틴을 정했다 (아침 산책, 50분 작업 + 10분 휴식)
- [ ] 번아웃 예방 계획이 있다

**Phase 5 완료 기준:** 자동화가 작동하고, 커뮤니티가 운영되고 있다
